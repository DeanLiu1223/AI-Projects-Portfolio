# 📚 課程作業集錦 (Course-Assignments)

這個 Repository 彙集了我在大學期間，在 **人工智慧、機器學習、深度學習** 等相關課程中所完成的程式作業與專案。

這些專案展示了我在 **資料分析、模型建構、演算法實作** 方面的能力與學習成果。

每個資料夾代表一門課程，內含專案筆記本 (`.ipynb`) 或程式碼檔案 (`.py`)，以及相關資料集。
